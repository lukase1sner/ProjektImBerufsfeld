# Projekt im Berufsfeld – Wintersemester 2025/26

In diesem Repository befinden sich alle Unterlagen zum **Projekt im Berufsfeld** in Zusammenarbeit mit der Firma **Viadee**.

---

## Projektziel

Ziel ist die Entwicklung einer **Quiz-App**, mit der Studierende Vorlesungsinhalte effizient wiederholen können.

Die Anwendung unterstützt verschiedene **Rollen**:

- Administrator  
- Quiz-Entwickler  
- Anwender  

Zentrale Funktionen der App sind:

- automatische **Fragengenerierung mithilfe von KI**
- **Feedback** zu beantworteten Fragen
- **Speicherung des Lernfortschritts**
